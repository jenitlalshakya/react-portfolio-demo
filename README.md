# React Portfolio Demo

A simple **React portfolio demo** project showcasing a personal portfolio website layout.  
This project is built with **React.js** and is intended for learning and demonstration purposes.

---

## Demo

You can view the live demo here:  
[https://jenitlalshakya.github.io/react-portfolio-demo](https://jenitlalshakya.github.io/react-portfolio-demo)

---

## Features

- Built using **React.js**
- Responsive layout for desktop and mobile
- Navigation bar with smooth scrolling
- Sections for About, Projects, and Contact (can be customized)
- Easy to deploy using **GitHub Pages**
